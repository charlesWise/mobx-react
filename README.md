###  技术栈
react + react-router + axios + mobx + mobx-react + antd-mobile


## 目录结构
```
react-cnode/
   |
   ├──icons/                    * 用到的icon
   |
   ├──components/               * 可复用组件
   │
   │──pages/                    * 展示的页面
   |
   ├──store/                    * mobx
   │
   ├──router                    * 路由
   |
   ├──routes/                   * 路由的配置，动态加载
   │
   │──axios/                    * axios的配置文件
   │
   │──utils/                    * 工具函数
   │    |
   |    |—— db.js               * localstorage操作
   |    |
   |    |__ index.js            * 别的工具函数
   │
   │ 
   │——constants/                * 一些常量
   │
   │
   │
   │──index.css                 * 全局的样式
   │__index.js                  * 入口文件
```
